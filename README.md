## SQL Commands Overview

Here is an overview of the different categories of SQL commands:

| **Category** | **Commands**                       |
|--------------|------------------------------------|
| **DDL**      | CREATE, DROP, ALTER, TRUNCATE      |
| **DML**      | INSERT, UPDATE, DELETE, MERGE      |
| **DCL**      | GRANT, REVOKE                      |
| **TCL**      | COMMIT, ROLLBACK, SAVEPOINT        |
| **DQL**      | SELECT                             |
